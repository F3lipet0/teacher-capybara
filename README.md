# 🦫 The Professor Capivara

Projeto desenvolvido para a disciplina de **Algoritmos e Programação de Computadores (APC)** do curso de **Ciência da Computação da UFAL - Campus Arapiraca**.

## 📖 Sobre o jogo

*The Professor Capivara* é um jogo educativo voltado para crianças!

No jogo, uma capivara precisa atravessar um rio cheio de predadores, como onças, jacarés, serpentes, gaviões e piranhas. Para conseguir avançar, o jogador deve responder corretamente às perguntas apresentadas.

As perguntas abordam conteúdos simples e educativos, como:

* Matemática básica;
* Gramática básica;
* Curiosidades sobre animais.

Cada resposta correta ajuda a capivara a continuar sua travessia. Caso o jogador erre 3 perguntas, ele perde.

---

## 🎯 Objetivo

O principal objetivo deste projeto foi colocar em prática os conteúdos estudados durante o semestre na disciplina de APC, desenvolvendo um jogo simples, e educativo.

---

## 🛠️ Tecnologias utilizadas

* Python 3
* Pygame

## 💻 Conceitos de programação utilizados

Durante o desenvolvimento do projeto foram utilizados diversos conceitos estudados em sala de aula, tais como:

* Variáveis;
* Tipos de dados;
* Estruturas condicionais (`if`, `elif`, `else`);
* Estruturas de repetição (`for` e `while`);
* Funções;
* Listas;
* Tuplas;
* Dicionários;
* Manipulação de arquivos;
* Entrada e saída de dados;
* Bibliotecas.

## ▶️ Como executar

### 1. Clone o repositório

```bash
git clone (https://github.com/F3lipet0/teacher-capybara.git)
```

### 2. Entre na pasta do projeto

```bash
cd the_professor_capivara
```

### 3. Instale as dependências

```bash
pip install pygame
```

### 4. Execute o jogo

```bash
python3 jogo_pygame.py
```
